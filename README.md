# 📷 사진 이벤트 자동 정리기

흩어진 수천~수만 장의 사진을 **사람이 기억하는 이벤트 단위**로 자동 정리한다
(`2021-09-17_175628`). 거창한 AI 없이 **촬영시각의 시간 간격**만으로 이벤트를
잘라낸다 — 몰아 찍다 몇 시간 끊기면 거기가 이벤트 경계.

> 핵심 철학: **사진은 두 번 못 찍는 데이터 → 데이터 손실 절대 불가.**
> 원본은 절대 안 건드리고, 복사만 하며, 사본마다 sha256으로 검증한다.

## 빠른 시작

```bash
brew install exiftool          # 메타데이터 추출(필수)
python3 -m venv .venv && .venv/bin/pip install Pillow pillow-heif
```

**웹 앱 (추천 — 터미널 불필요):** Finder에서 `사진정리.command` 더블클릭
→ 브라우저에서 폴더 선택 → 미리보기 → 복사 → 유사 사진 정리까지 한 화면에서.

**CLI:**
```bash
.venv/bin/python organize.py <원본폴더> <목적지> --dry-run   # 계획만 출력
.venv/bin/python organize.py <원본폴더> <목적지> --copy      # 실제 복사
.venv/bin/python organize.py <목적지> --review-similar       # 유사 사진 정리
```

## 무엇을 하나

1. **촬영시각 추출** — exiftool 일괄 호출. 사진은 `DateTimeOriginal`, 영상은
   `CreationDate`. 시각 없거나 깨진 값은 `_분류안됨/`으로 격리(파일시각으로
   추정하지 않음).
2. **라이브포토 짝 묶기** — `IMG_1234.HEIC` + `.MOV`를 한 덩어리로(절대 안 갈라짐).
3. **이벤트 분할** — 4시간(`--gap-hours`) 넘게 끊기면 새 이벤트.
4. **쓰레기 거름** — 스크린샷·배경화면·아트워크 등 이름 패턴은 `_기타/`로
   (진짜 사진은 EXIF가 없어도 날짜로 정리).
5. **안전 복사** — `.tmp`에 쓰며 sha256 계산 → fsync → 되읽어 검증 → 원자적 rename.
   재실행해도 중복 0(멱등). SQLite 맨페스트에 모든 복사를 기록.
6. **유사 사진 정리** — 비슷한 묶음에서 맘에 드는 것만 고르면 나머지를
   `_삭제예정/`(휴지통)으로 **이동**. 영구삭제는 사용자가 직접.

## 안전장치

- 원본 무손상 · dry-run 무쓰기 · 멱등 — 테스트로 보증 (`pytest`).
- 복사 후 sha256 검증(`--no-verify`로 해제 가능).
- 여유공간 사전 점검, exFAT 4GB 한계 격리.
- **자동 삭제 없음.** 유사 정리도 휴지통 이동까지만.
